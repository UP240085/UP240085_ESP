## Circuito Blink
![image alt](https://github.com/UP240085/UP240085_ESP/blob/40ec239c7a00c3b7b995fa44269dc6b969ac2791/Circuito%201.jpg)

El objetivo de esta práctica fue hacer una señal de SOS en código morse con el LED con una duración entre señal de 1seg; además se consideró que los puntos cortos tiene una duración de 200ms y el punto largo de 500ms. El programa se vió refeljado en un circuito con LED externo, el cuál mostró la señal SOS con éxito
